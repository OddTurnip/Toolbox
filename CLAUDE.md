# Utility Tools

Use `template.html` as a starting point for new tools. One HTML file per utility.

## UI/UX Guidelines

**Minimize vertical space:**
- Use two-column form layouts (labels left, controls right) instead of stacked headers
- Keep padding and margins tight - start small, user will ask for more if needed
- Combine related controls on the same row when possible (e.g., slider + number input)

**Avoid layout displacement:**
- Use tooltips/popups for transient feedback (e.g., "Copied!") instead of status banners that push content down
- Prefer inline error messages within existing UI elements over separate error containers

# OddTurnip Toolbox

A collection of simple, browser-based utility tools. All tools run entirely client-side with no server dependencies.

Open `index.html` in a browser, access the individual tools directly, or deploy to any static web host.

## Tools

- **Word Count** - Count words, sentences, and paragraphs
- **Character Count** - Analyze character frequency and detect invisible characters
- **Text Diff** - Compare two texts and highlight differences
- **ROT13** - Simple letter substitution cipher
- **Mirror Text** - Flip text backwards or upside down
- **Zalgo Text** - Create glitchy, corrupted text
- **Telegram Sticker Maker** - Create 512px stickers with text overlay

## Tech Stack

- Plain HTML/CSS/JavaScript
- No build step, no dependencies
- Shared styles in `styles.css`
- Template for new utilities in `template.html`
